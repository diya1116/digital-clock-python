⏰ Python Desktop Digital Clock

This is a clean and modern digital clock application that runs straight on your computer desktop. I built it using Python to practice creating app interfaces, handling real-time data, and coding simple interactive buttons.

⭐ Cool Features
1. Real-time Time & Date: Displays the exact time, day of the week, and date accurately.
2. 12-Hour / 24-Hour Switch: Tap a button to instantly switch between normal time (AM/PM) and military time.
3. Custom Color Themes: Change the look of the app instantly using the built-in sidebar menu
4. Smooth Performance: The code updates the time silently in the background without lagging your computer.

🛠️ Tools Used:
Python: The main programming language.
Tkinter: The built-in Python tool used to design the windows, layout, and buttons.

🚀 How to Run It on Your Computer:
1. Download the code files to your computer.
2. Open your computer's terminal or command prompt inside that folder.
3. Type this command and press Enter to start the clock: python clock.py
